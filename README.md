# Ticketautomat 🎫

A ticket vending machine written in Java ☕ by [Vincent](https://github.com/VinceDerPrince), [Ruben](https://github.com/rubenkol) and Caspar.
Build with the Vaadin Flow framework and served on spring boot 🍀.
